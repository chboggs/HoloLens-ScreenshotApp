## Deploy Instructions:
* Download Repo
* Open Alpha_Build Folder
* Open ScreenshotTool.sln with Visual Studio
* Set build settings to "Release", "x86", "Device" (if hololens is plugged into usb)
* Hit "Build">"Deploy ScreenshotTool"
